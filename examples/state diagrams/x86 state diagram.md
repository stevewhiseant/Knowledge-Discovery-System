## ⭐ 2. STATE DIAGRAM — s9 (Bresenham Rasterizer)
Implements the core line‑drawing loop

Code
                     +----------------+
                     |  START (s9)    |
                     +--------+-------+
                              |
                              v
                +---------------------------+
                | Compute dx, dy            |
                | dx = y2 - y1              |
                | dy = x2 - x1              |
                +-------------+-------------+
                              |
                              v
                +---------------------------+
                | Initialize Bresenham vars |
                | d06020..d0602e, bx, count |
                +-------------+-------------+
                              |
                              v
                     +----------------+
                     |  LOOP_START    |
                     +--------+-------+
                              |
                              v
                     +----------------+
                     |    call s7     |
                     +--------+-------+
                              |
                              v
                +---------------------------+
                | Check bx < 0              |
                +-------------+-------------+
                              |
               +--------------+--------------+
               |                             |
             Yes                             No
               |                             |
               v                             v
   +------------------------+     +------------------------+
   | NEG_PATH:              |     | POS_PATH:              |
   | si += d06028           |     | si += d06022           |
   | di += d0602a           |     | di += d06020           |
   | bx += d0602c           |     | bx += d0602e           |
   +-----------+------------+     +-----------+------------+
               |                             |
               +--------------+--------------+
                              |
                              v
                +---------------------------+
                | Decrement count           |
                | If count > 0 → LOOP_START |
                +-------------+-------------+
                              |
                              v
                     +----------------+
                     |    S9_DONE     |
                     +----------------+
## ⭐ 3. STATE DIAGRAM — s7 (Pixel Plotting)
Computes offset, applies mask, writes pixel

Code
                   +----------------+
                   |  START (s7)    |
                   +--------+-------+
                            |
                            v
                +---------------------------+
                | Compute offset =          |
                |   calculate_offset(di,si) |
                +-------------+-------------+
                              |
                              v
                +---------------------------+
                | mask = d06010[si & 0x03]  |
                +-------------+-------------+
                              |
                              v
                +---------------------------+
                | old = es[offset]          |
                | new = (old & mask) | AH   |
                +-------------+-------------+
                              |
                              v
                +---------------------------+
                | es[offset] = new          |
                +-------------+-------------+
                              |
                              v
                     +----------------+
                     |   S7_DONE      |
                     +----------------+
